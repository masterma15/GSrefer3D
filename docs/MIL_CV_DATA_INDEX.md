## 4. 技术要点（被追问时用）

### 成像仿真

| 模块 | 要点 |
|------|------|
| 目标 | 五类舰 3D + LOD；BRDF/亮温；Kelvin/热尾迹 |
| 背景 | 0–5 级海况；云 0–30%；海陆 MODTRAN |
| 大气 | 辐射传输；湍流/气溶胶 MTF |
| 载荷 | 光学/探测器/电路 MTF；像移；噪声；量化 |
| SAR | 回波成像；与光学配准 |

### 检测（YOLOv8）

| 项 | 内容 |
|------|------|
| 类别 | 航母、两栖攻击舰、巡洋舰、驱逐舰、民船 |
| 尺度 | 大/小面目标、点目标（分模型） |
| 数据 | 仿真图 → 标注 → 训练/推理 → predictions.csv |
| 融合 | 光学粗定位 → SAR 细提取 → 决策级融合 |
| 结项 | 2025-09 多目标 / 多分辨率用例通过 |

**面试介绍**（与 `RESUME_AND_INTERVIEW.md` 同步）：先讲清成像→数据→YOLOv8 检测→融合定位闭环，再讲自己负责的数据接入、联调、结项测试。
