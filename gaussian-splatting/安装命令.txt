使用git克隆代码（打开cmd d盘根目录）
git clone https://github.com/graphdeco-inria/gaussian-splatting --recursive




-------------------------环境配置方法一：

在conda中 打开bast（root环境）

切换到高斯泼溅工作目录
cd /d D:\gaussian-splatting

以此执行下面下面三条命令
SET DISTUTILS_USE_SDK=1
conda env create --file environment.yml
conda activate gaussian_splatting







-------------------------环境配置方法一：
在conda navigator 图形界面中导入  D:\gaussian-splatting目录下 environment.yml文件并等待配置完成
运行新导入的环境（open terminal）


切换到高斯工作目录：
cd /d D:\gaussian-splatting

以此执行下面六行命令配置依赖模块：
cd  D:\gaussian-splatting\submodules\diff-gaussian-rasterization
python setup.py install

cd  D:\gaussian-splatting\submodules\simple-knn
python setup.py install

cd  D:\gaussian-splatting\submodules/fused-ssim
python setup.py install


安装opencv（如果出现过pip报错，需要执行该命令）
pip install opencv-python
