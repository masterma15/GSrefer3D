
-------------------------训练模型：

列出所有环境
conda env list

切换到高斯环境（后面改成自己的环境名称）
conda activate envGS


打开工作目录
cd /d D:\gaussian-splatting

处理手机拍摄图片的某些图像的EXIF方向标签异常或不支持问题
magick mogrify -strip data2/input/*.jpg

预处理图片
python convert.py -s data2

训练模型
python train.py -s data2 -r 4

渲染模型
python render.py -m output\文件名 --custom_views --output_path 输出文件夹名称

查看模型
E:\gaussian-splatting\viewers\bin\SIBR_gaussianViewer_app.exe -m E:\gaussian-splatting\output\6f4fa6f4-6
